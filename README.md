# QuickBite – Food Ordering Website

QuickBite is a simple **Food Ordering Website** built using **HTML, CSS, and JavaScript**.  
The entire project is implemented in a **single file (`index.html`)**, making it easy to run and understand.

This project was developed for a **Web Development Competition**, focusing on UI design, layout, and basic client-side functionality.

---

## Features

- Navbar with cart count  
- Hero section  
- Food menu displayed using cards  
- Add to Cart functionality  
- Checkout and simulated payment flow  
- Login system (client-side validation)  
- Dark mode toggle  
- Fully interactive UI without backend

## Payment Flow

- Add items to cart  
- Click **Checkout & Pay**  
- Select payment method (UPI / Debit Card / Cash on Delivery)  
- Payment is simulated (no real gateway used)
